# ENDF Fortran test

These tests are intended for ENDF integer/float numbers parsing examples.
Internal `nkl::data::endf::parse_endf_integer()` and
`nkl::data::endf::parse_endf_float()` must have the same behavior.

## Usage

```console
# edit float.f90 or integer.f90
$ make
$ ./integer
$ ./float
```
